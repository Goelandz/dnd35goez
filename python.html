Installing the application

All my small DM tool use Python. How to use it? Fairly simple.

  • Head out to https://www.python.org/ to download it and install it.
  • On windows search for "Edit environment variables for your account"
  • Select the variable named "Path", then click "Edit"
  • Click on "New"
  • On the requested path inscribe "%USERPROFILE%\AppData\Local\Programs\Python\Launcher\"
  • Click on "Ok"
  • Click on "Ok"
